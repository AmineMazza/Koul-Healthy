## Install

Clone repo

```
git clone https://github.com/AmineMazza/Koul-Healthy.git
```

composer update/install

```
composer install
```

NPM dependencies

```
npm install
```

Générez une nouvelle clé de cryptage

```
php artisan key:generate
```

Configurez votre base de données

```
nano .env
```

Exécutez les migrations

```
php artisan migrate
```
